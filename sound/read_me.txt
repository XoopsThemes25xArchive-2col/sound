The Sound theme is a 2 column theme for XOOPS 2.0.X series (including 2.0.15)
the right blocks have been added to the left blocks

This theme is a limited run so please save it. you might not be able to download it again.




This package also includes a screenshot, and two templates for the "system" module :

- system_redirect.html
- system_siteclosed.html

~note: To use these you must rename the Sound theme to Default. ~


This theme is a valid W3C CSS and XHTML with a basic XOOPS installation
